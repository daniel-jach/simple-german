Wie finde ich alle Texte auf dieser Internet-Seite? Oder Sie können in der Mitte der Seite auf die Bilder klicken. Klicken Sie zum Beispiel auf das Bild: Schwere Wörter verstehen Klicken Sie auf das schwere Wort. Das schwere Wort wird in Leichter Sprache erklärt. Zurück zum Text: Klicken Sie auf Fenster schließen X. Die Pfeile benutzen Auf jeder Seite sind unten zwei blaue Pfeile: Auf den Pfeil nach links klicken: Sie kommen zurück auf die Seite, wo Sie zuletzt waren. Auf den Pfeil nach oben klicken: Sie kommen nach oben auf der Seite, auf der Sie jetzt sind. Die Schrift größer oder kleiner machen A+: Dieser Knopf macht die Schrift größer. Er ist immer rechts. Klicken Sie den Knopf, dann wird die Schrift größer. 1 mal klicken macht die Schrift etwas größer. 3 mal klicken macht die Schrift sehr groß. A-: Dieser Knopf macht die Schrift kleiner. Klicken Sie den Knopf, dann wird die Schrift kleiner. 1 mal klicken macht die Schrift etwas kleiner. 3 mal klicken macht die Schrift sehr klein. Den Text vorlesen lassen Der Lautsprecher ist der Knopf für das Vorlesen. Er ist immer rechts neben der Überschrift: So wird ein Text vorgelesen: Klicken Sie auf den Knopf. Dann öffnet sich ein Fenster und der Text wird vorgelesen. Sie können das Fenster bedienen wie einen CD-Spieler: Zurück Stopp Start Das Fenster schließen: Klicken Sie oben rechts auf X. Dann schließt das Fenster und Sie sind wieder im Text. Achtung: Vielleicht öffnet sich das Fenster und es wird nichts vorgelesen. Dann fragen Sie bitte jemand, der sich sehr gut mit Computern auskennt. Den Test machen Auf manchen Seiten ist ganz unten ein Test. Neben dem Test ist immer dieses Bild: ein Fragebogen. Mit dem Test können Sie selber prüfen: Habe ich den Text verstanden. Wollen Sie den Test machen? Dann klicken Sie auf den Knopf Test starten. Wissen Sie die Antwort, dann klicken Sie in den Kreis neben dieser Antwort. Dann Klicken Sie auf den Knopf Fertig. Jetzt können Sie die richtige Antwort lesen. Ein Wort suchen Die Lupe ist der Knopf für das Suchen. Die Suche ist immer links oben: Schreiben Sie Ihr Such-Wort neben die Lupe. Dann klicken Sie auf die Lupe. Es kommt eine Liste mit Texten. Sie können jetzt: Einen Text lesen. Dazu müssen Sie auf die Überschrift von dem Text klicken. Sie können auch: Weiter suchen. Dazu müssen Sie ganz unten auf der Seite auf ein blaues Wort klicken. Bei manchen Such-Worten kommt keine Liste mit Texten. Dann gibt es keine Texte mit dem Such-Wort. Aber da stehen blaue Worte. Für die blauen Worte gibt es Texte. Sie können auf ein blaues Wort klicken und den Text lesen. Einen Text drucken Der Drucker ist der Knopf für das Drucken. Er ist immer rechts. Klicken Sie auf diesen Knopf. Dann kommt ein neues Fenster. Klicken Sie da auf den Knopf Drucken. Dann wird der Text gedruckt. Was mache ich, wenn ich mit der Internet-Seite nicht klarkomme?
